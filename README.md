# Trove

Trove is an e-commerce application that includes a web and a mobile interface. This repo consists of the web application. The end users of the web are the admin, vendor, and the Customer Service Representative.

## To run the Web APIs

Make sure you have .NET SDK 8.0 installed locally on your machine. Then go to the 'backend' directory and run the following commands:

### dotnet restore

### dotnet build

### dotnet run

This will run the backend of the application. You can now test the APIs using a testing tool like Postman
